# MerrBio – Farmers Marketplace Platform

A full-stack marketplace platform developed during a hackathon that connects farmers directly with buyers, enabling them to showcase, promote, and sell agricultural products online.

The platform aims to simplify the agricultural supply chain by creating a direct connection between producers and consumers.

---

## Overview

MerrBio is a web-based marketplace designed to support local farmers by providing a digital platform where they can list their products and reach potential buyers without intermediaries.

The platform allows farmers to publish product listings, while customers can browse available products and connect directly with sellers.

Developed within a hackathon environment, the project focused on rapid development, usability, and practical real-world impact.

---

## Features

### Farmer Features

* Create product listings
* Display product details
* Manage available products
* Showcase agricultural goods

### Buyer Features

* Browse available products
* View product information
* Search for products
* Contact farmers directly

### Platform Features

* Responsive user interface
* Dynamic product catalog
* User-friendly navigation

## Hackathon Project

This project was developed during a hackathon, where the primary objective was to rapidly design and implement a practical solution for challenges in the agricultural sector.

The focus was on delivering a functional MVP (Minimum Viable Product) within a limited timeframe while maintaining usability and scalability.
